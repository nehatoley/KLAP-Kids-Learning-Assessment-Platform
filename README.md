KLAP is an intelligent learning and assessment platform designed for children. The system leverages YOLOv8-based object detection to identify real-world objects through a webcam and create an interactive educational experience.

Features
Real-time object detection using YOLOv8
Interactive learning mode for object recognition
Assessment and quiz module for evaluating learning outcomes
Voice-based feedback for detected objects
Parent and teacher dashboards for progress monitoring
Performance tracking and reporting

Technologies Used

Python

YOLOv8 (Ultralytics)

OpenCV

Flask

SQLite/MySQL

pyttsx3 (Text-to-Speech)
